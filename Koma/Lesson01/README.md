## 课程介绍

webpack 是一个开源的 Web 项目打包工具，主要用于 Javascript 项目，并可以包括 HTML,CSS,图片资源等前端功能项目资源。webpack 帮我们解决的项目模块之间的依赖关系，自动生成项目使用的静态模块。
